Main클래스 실행시키게 되면 로그인 화면이 나타나게 된다.
여기서 회원가입 버튼을 누르게 되면 회원가입 창이 뜨게 되는데 여기서 정보들을 입력하여 회원가입을 진행한다.
회원가입이 끝나면 다시 로그인 화면창이 뜨게 되는데 여기서 두 가지 접근이 가능하다.
회원가입을 한 정보로 로그인을 하게 되면 리뷰 프로그램을 사용할 수 있다.
관리자로 접근하고 싶으면 학번에 administrator 비밀번호에 0000을 입력해서 들어가게 되면 회원가입된 user들의 정보가 나오게 된다.
회원가입을 한 User로 접근했을 경우에는 여러가지 기능을 사용할 수 있다.
1.리뷰 추가
-리뷰 추가 버튼을 누르면 리뷰를 추가할 수 있는 frame이 뜨게 되는데 여기서 양식들을 모두 입력하고 추가 버튼을 누르면 리뷰가 추가된다.
2.리뷰 조회
-이렇게 리뷰를 등록하고 리뷰 조회 버튼을 누르면 화면에 등록한 식당들과 리뷰들이 나오게 된다.
3.리뷰 수정
-화면에 뜬 리뷰중에서 하나를 선택하여 리뷰 수정 버튼을 누르면 리뷰 수정이 가능한 frame이 뜨게 되고 여기서 수정해서 저장 버튼을 누르면 리뷰가 수정이 된다.
4.카테고리, 식당 검색
-frame의 위에 보면 식당 이름을 검색할 수 있는 텍스트 입력 필드가 있다. 그곳에 식당 이름의 일부를 입력하여 검색 버튼을 누르면 식당의 리스트들이 나오게 된다. 또한 카테고리 중 하나를 선택하여 검색 버튼을 누르면 해당되는 카테고리의 식당들이 모두 나오게 된다. 이것을 할 때에는 중복되게는 사용할 수 없고 둘 중 하나의 기능만 사용가능하다.
5.식당 추천
-식당 추천 버튼을 누르게 되면 이곳에 등록된 식당 중에서 하나의 식당이 랜덤으로 뜨게 된다.
